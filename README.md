# Quiz-Game
Es handelt sich um ein Quiz Spielchen.

# Features:

1 - Zeitbeschränkung:
Der Spieler hat nut 10 Sekunden, um eine Frage zu beantworten.
Dafür wurde ein Time-Progress-Bar implementiert.

2 - Hint:
Die Hint-Taste gibt der Spieler die Möglichkeit, zwei falsche Antworten zu vermeiden.
Dies ist aber nur einmal pro 5 Fragen anwendbar.

3 - Musik:
Hintergrundmusik, die gestummt werden kann.

4 - Highscore Schwarzesbrett:
Eine Ergebnistabelle mit aller vergangenen Spielzüge wird erstellt.

5 - Richtige Antwort:
Nach jede Frage wird die richtige Antwort für kurze Zeit mit grün gefärbt.
